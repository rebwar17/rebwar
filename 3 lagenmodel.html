3 lagen model

Er zijn drie lagen de fysieke, logische en toepassingslaag

De fysieke laag

In deze laag zit de hardware van het apperaat van moederbod, de accu en het scherm.

De logische laag

De logiscge laag is de software die de hardware aanstuurt. Deze software zorgt dat apps op het apperaat uitgevoerd worden. Het bestaat voornamelijk uit besturingssystemen.

De toepassingslaag

Bestaat uit software bijvoorbeeld apps die je kan downloaden.
